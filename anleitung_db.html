Anleitung Stempeluhr-Datenbank

Einloggen

Gehen Sie auf http://www.pmd.li/stempeluhr/adminer.php und loggen Sie sich mit folgenden Daten ein:

Server: localhost
Benutzer: web233
Passwort: haben berechtige Personen per Mail erhalten
Datenbank: usr_web233_7


Es erscheint nun eine Liste mit allen Tabellen (inkl. automatisch generierten Tabellen, sogenannte "Views")



Mitarbeiter

Wechseln Sie zur Tabelle "mitarbeiter" und tragen Sie einen neuen Mitarbeiter ein:

Mitarbeiter_id: leer lassen, wird automatisch generiert
Nachname/Vorname: Nach- resp. Vorname des Mitarbeiters. Name muss eindeutig sein.
Monatslohn: 1 = Ist im Monatslohn angestellt und erhält Samstagszulage; 0 = keine Samstagszulage


Wechseln Sie nun zur Datenansicht unter "Daten auswählen". Dort sehen Sie in der spalte "mitarbeiter_id" welche ID dem eingetragenen Mitarbeiter zugeordnet wurde. Wenn ein Zeiteintrag oder Arbeitsort dieses Mitarbeiters in der entsprechenden Tabelle geändert werden soll, brauchen Sie diese ID um auf diesen Mitarbeiter zu verweisen.


Filialen

Wechseln Sie zur Tabelle "filialen" dort können Sie die ID für jede Filiale unter "filialen_id" entnehmen. Klicken Sie auf "Daten auswählen" um sich alle Filialen anzeigen zu lassen.

Arbeitsort

Hier können Sie eintragen welcher Mitarbeiter ("mitarbeiter_id") in welcher Filiale ("filialen_id") arbeiten kann. Es werden beim Anmelden in einer Filiale nur Mitarbeiter zur Auswahl stehen, deren Mitarbeiter-ID in dieser Tabelle für die entsprechende Filialen-ID einen Eintrag haben.

Zeitstempel

Hier können Sie Zeiteinträge manuell hinzufügen, entfernen oder ändern. Achten Sie auf die korrekten Mitarbeiter- und Filialen-IDs. "zeitstempel_id" und "last_updateted" werden automatisch gerneriert. Müssen also leer gelassen werden.
